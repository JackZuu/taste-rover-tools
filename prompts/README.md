# SmarTR Food — AI Prompts

This folder documents every prompt sent to OpenAI (gpt-4o-mini) by the backend.
Each file describes the intent, the exact prompt template, expected response format,
and notes on how the output is validated and used.

| File | Module | Purpose |
|------|--------|---------|
| [seasonal.md](seasonal.md) | Seasonal Foods | UK ingredients in season for a given month |
| [celebrations.md](celebrations.md) | Upcoming Events | UK food-relevant celebrations in the next 90 days |
| [regional.md](regional.md) | Regional Demand | Consumer demand insights by UK region |
| [nutrition.md](nutrition.md) | Nutrition Sample | Calorie & macro estimate for a street-food serving |
| [menu_generator.md](menu_generator.md) | Menu Proposal | Final curated menu given weather, trends, region, etc. |
